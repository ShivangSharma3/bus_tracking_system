🎨 UI Features

  • 🔵 Blue Card: Distance to current stop with pin icon (📍)
  • 🟢 Green Card: Distance to next stop with target icon (🎯)
  • 📏 Smart Formatting: Shows "150m" for short distances, "2.34km" for longer ones
  • 📱 Responsive Grid: Two columns on desktop, stacks on mobile
  • ✨ Conditional Display: Only shows when distance data is available
